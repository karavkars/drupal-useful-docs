Useful contributed modules

  1. Classy paragraph - Add classes to paragraph + to filter class use views entity reference view
  2. Field Formatter class - Adds class on fields in view mode
  3. Fields group - Add grouping around fields in display mode
  4. Panels - gives flexibility to build complex displays
  5. Paragraph View mode - add view mode selection on paragraph while authoring content
  6. Style guide - displays styles used in theme
  7. Twigtweak - dump variable in twig file
  8. Vardumper - dump variable in php drupal.
  9. Views export data - Export views results to csv, xls, json, etc.
  10. Views Schema - Generate code to include custom table in drupal views
